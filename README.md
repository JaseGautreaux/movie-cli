# Movie CLI

This is a command line app that searches for any movie and
displays the movies details using the TMDB API.

## How to run

1. Clone this repo
2. Install 'pip install request'
3. Sign up for free at themoviedb.org and get your free API key
4. Replace ENTER YOUR KEY HERE with your API key
5. Then run 'python movie.py'
